Word Guess Game

Getting Started

Welcome to the Word Guess Game! This game mimics the famous Hangman game (minus the hangman image).

The theme for this game is colours of the rainbow! There are 7 colours of the rainbow, but this game only reveals 3 rainbow colours, so which colour word would appear? Only one way to find out! Play the game!

Please read the instructions below to learn how to play the game.

Instructions

  1. Press any key to start the game
  2. Press a letter on your keyboard (note: only lowercase letters allowed) to begin guessing the word
    • If the letter is contained in the word, the letter will appear; otherwise, you will have to try again. You are allowed multiple tries.
    • Once all the letters of the word have appeared, you have won the game and will receive a score of 1.
  3. Click the Play again button to start a new game.

Enjoy the game!
